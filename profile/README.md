# NexVigilant

**Empowerment Through Vigilance**

Open-source pharmacovigilance infrastructure. 280 Rust crates, 1,800+ MCP tools, sub-microsecond decision trees.

## What We Build

| Project | Description |
|---------|-------------|
| [nexcore](https://github.com/nexvigilant/nexcore) | The Vigilance Kernel — 280 Rust crates for signal detection, causality assessment, and regulatory intelligence |
| [NexVigilant Station](https://github.com/nexvigilant/nexvigilant-station) | 1,800+ free pharmacovigilance MCP tools at mcp.nexvigilant.com |
| [rsk-core](https://github.com/nexvigilant/rsk-core) | Microgram decision tree runtime — atomic self-testing programs for PV workflows |
| [NexVigilant Radio](https://github.com/nexvigilant/nexvigilant-radio) | Receiver app for Station tools |
| [Crystalbook](https://github.com/nexvigilant/crystalbook) | Eight Laws of System Homeostasis — governance framework |

## For Developers

- **Rust crates**: `cargo add nexcore-pv-core` — signal detection (PRR/ROR/IC/EBGM), causality (Naranjo/WHO-UMC)
- **MCP tools**: Connect to `mcp.nexvigilant.com` — no auth required, 1,800+ tools
- **Decision trees**: Sub-microsecond PV classification via microgram programs
